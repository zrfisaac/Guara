# <img src="Guara.ico" alt="Logo" width="64" height="64"> Guara - The Ultimate Database Management Tool

## UNDER DEVELOPMENT!

Guara is a comprehensive, versatile database management tool that simplifies complex database operations. Designed for developers and data managers, Guara supports a variety of databases and offers an extensive set of features to boost productivity, streamline workflows, and enhance data handling capabilities.

## Key Features

- **Data Export and Import:** Effortlessly export and import data across multiple formats, including CSV and Excel.
- **SQL Script Execution:** Run SQL scripts directly within Guara, streamlining the management of large databases.
- **Data Conversion:** Convert data seamlessly between different formats, such as CSV to Excel and vice versa.
- **Multi-Database Support:** Compatible with Firebird, Oracle, SQL Server, MariaDB, PostgreSQL, MySQL, MongoDB, and more.
- **User-Friendly Interface:** Built with ease of use in mind, Guara provides a clean, intuitive interface.

## Compatibility

Guara is compatible with a wide range of databases and data formats, ensuring adaptability to various development environments and database systems.

Supported databases:

- Firebird
- Oracle
- SQL Server
- MariaDB
- PostgreSQL
- MySQL
- MongoDB
- CSV and Excel data formats for added flexibility

## Installation

To get started with Guara, download the latest release and follow these steps:

- Download the Guara installer from the release page.
- Run the installer and follow the on-screen instructions.
- Open Guara, connect to your database, and start managing data more efficiently.

## Usage

- **Connecting to a Database:** Open Guara and select the database you wish to connect to. Configure the connection settings and click "Connect."
- **Exporting Data:** Select the table or query result you want to export, choose the desired format (CSV or Excel), and click "Export."
- **Importing Data:** Choose your data source, configure import settings, and map data fields as needed before importing.
- **Running SQL Scripts:** Access the SQL editor, write or load your SQL script, and click "Run" to execute.
