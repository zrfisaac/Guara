# <img src="Guara.ico" alt="Logo" width="64" height="64"> Guara - A Ferramenta Definitiva de Gestão de Bancos de Dados

Guara é uma ferramenta abrangente e versátil de gestão de bancos de dados, projetada para simplificar operações complexas. Com recursos poderosos, Guara foi feito para desenvolvedores e gestores de dados que buscam uma solução que aumente a produtividade, otimize fluxos de trabalho e melhore o manuseio de dados.

## Principais Funcionalidades

- **Exportação e Importação de Dados:** Exporte e importe dados em diversos formatos, incluindo CSV e Excel, de forma simples.
- **Execução de Scripts SQL:** Execute scripts SQL diretamente no Guara, facilitando a gestão de grandes bancos de dados.
- **Conversão de Dados:** Converta dados facilmente entre formatos como CSV e Excel.
- **Compatibilidade com Múltiplos Bancos de Dados:** Suporte para Firebird, Oracle, SQL Server, MariaDB, PostgreSQL, MySQL, MongoDB e mais.
- **Interface Intuitiva:** Desenvolvido para ser fácil de usar, Guara oferece uma interface limpa e acessível.

## Compatibilidade

Guara é compatível com uma ampla gama de bancos de dados e formatos de dados, garantindo flexibilidade para diversos ambientes de desenvolvimento e sistemas de bancos de dados.

Bancos de dados suportados:

- Firebird
- Oracle
- SQL Server
- MariaDB
- PostgreSQL
- MySQL
- MongoDB
- Formatos de dados CSV e Excel para maior versatilidade

## Instalação

Para começar a usar o Guara, baixe a versão mais recente e siga os passos abaixo:

- Baixe o instalador do Guara na página de releases.
- Execute o instalador e siga as instruções na tela.
- Abra o Guara, conecte-se ao seu banco de dados e comece a gerenciar dados de forma mais eficiente.

## Como Usar

- **Conectando a um Banco de Dados:** Abra o Guara e selecione o banco de dados que deseja conectar. Configure as informações de conexão e clique em "Conectar".
- **Exportando Dados:** Selecione a tabela ou o resultado da consulta que deseja exportar, escolha o formato (CSV ou Excel) e clique em "Exportar".
- **Importando Dados:** Escolha a origem dos dados, configure as opções de importação e faça o mapeamento de campos conforme necessário antes de importar.
- **Executando Scripts SQL:** Acesse o editor SQL, escreva ou carregue seu script SQL e clique em "Executar" para rodá-lo.
